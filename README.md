# 🛍️ Retail Sales Analysis using SQL

## 📌 Project Overview

This project analyzes retail sales data using **MySQL**. It includes data loading, indexing, exploratory data analysis (EDA), and business-focused SQL queries to gain insights into revenue, customers, retention, and products.

## 🛠️ Tools Used

* MySQL
* SQL

## 📂 Dataset

The project uses a retail sales dataset containing information such as:

* Customer ID
* Invoice Number
* Product Description
* Invoice Date
* Quantity
* Price
* Total Sales
* Country

## 🔍 Analysis Performed

### Revenue Analysis

* Total revenue generated
* Highest revenue month
* Month-over-month revenue growth
* Revenue decline analysis
* Contribution of top 10 customers

### Customer Analysis

* Top customers by revenue
* Average revenue per customer
* High-value customers
* One-time vs repeat customers
* Customer Lifetime Value (CLV)

### Retention Analysis

* Repeat purchase analysis
* Customers purchasing again within 30 days
* Customers inactive for the last 3 months
* Month-over-month retention rate

### Product Analysis

* Best-selling products by quantity
* Highest revenue-generating products
* Products with high sales but comparatively low revenue

## 🚀 SQL Concepts Used

* Common Table Expressions (CTEs)
* Window Functions (`RANK`, `LAG`, `CUME_DIST`, `NTILE`)
* Aggregate Functions
* CASE Statements
* GROUP BY and HAVING
* Index Creation
* Date Functions
* Joins
